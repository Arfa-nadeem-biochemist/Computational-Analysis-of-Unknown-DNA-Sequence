# Computational-Analysis-of-Unknown-DNA-Sequence
Bioinformatics analysis and functional annotation of the SARS-CoV-2 Spike Glycoprotein gene from an unknown DNA sequence.
# Computational Analysis of an Unknown DNA Sequence

# Project Overview

This project involved the comprehensive bioinformatics analysis of an unknown DNA sequence to determine its potential biological identity and function.

The analysis began with gene identification to locate potential coding regions within the sequence. Transcriptional features were then examined to investigate mRNA characteristics and regulatory elements. The translated protein was analyzed to determine its physicochemical properties and functional significance.

To understand evolutionary relationships, phylogenetic analysis was performed using homologous sequences. Protein–protein interaction analysis provided insights into possible functional partners, while subcellular localization prediction helped identify the most likely cellular compartment of the protein.

Overall, this project demonstrates an integrated bioinformatics workflow for the functional annotation and characterization of an unknown DNA sequence.

## Workflow

1. Gene Identification
2. mRNA Feature Analysis
3. Protein Prediction and Characterization
4. Physicochemical Property Analysis
5. Phylogenetic Analysis
6. Protein–Protein Interaction Prediction
7. Subcellular Localization Prediction
8. Functional Interpretation
   
# Bioinformatics Tools and Databases Used

- NCBI Database – Sequence retrieval and annotation
- NCBI BLAST – Homolog identification and sequence similarity analysis
- ORF Finder – Gene prediction and coding region identification
- ExPASy ProtParam – Physicochemical characterization of proteins
- ExPASy ProtScale – Protein property analysis
- Clustal Omega – Multiple sequence alignment
- MEGA – Phylogenetic tree construction and evolutionary analysis
- STRING – Protein–protein interaction prediction
- BioGRID – Functional interaction analysis
- CELLO – Subcellular localization prediction
- PSORT – Cellular localization analysis
  
# Key Outcomes

- Identification of potential coding regions within the DNA sequence
- Analysis of transcriptional and mRNA features
- Prediction and characterization of the encoded protein
- Evolutionary relationship assessment through phylogenetic analysis
- Prediction of protein interaction networks
- Determination of probable subcellular localization
- Functional annotation of an initially unknown biological sequence
